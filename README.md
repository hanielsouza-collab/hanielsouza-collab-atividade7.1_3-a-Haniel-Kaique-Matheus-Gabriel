## Descrição do Sistema
O JobConnect é um sistema Front-End desenvolvido para conectar candidatos a oportunidades de emprego.

O sistema permite que usuários realizem cadastro, login e visualizem suas informações profissionais de forma organizada e intuitiva.

---

## Tema Escolhido
Sistema de Vagas de Emprego

---

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- LocalStorage

---

## Estrutura das Páginas

### index.html
Página inicial do sistema contendo:
- nome do projeto
- menu de navegação
- banner
- descrição do sistema

---

### cadastro.html
Tela de cadastro contendo:
- nome
- e-mail
- senha
- área profissional
- cargo desejado

---

### login.html
Tela de autenticação contendo:
- e-mail
- senha

---

### sistema.html
Área principal do usuário contendo:
- nome
- email
- área profissional
- cargo desejado
- mensagem de boas-vindas
- logout

---

### usuarios.html
Tela para visualização dos usuários cadastrados.

---

### configuracoes.html
Tela de configurações do sistema.

---

## Funcionalidades do Sistema
- cadastro de usuário
- login de usuário
- logout
- armazenamento em LocalStorage
- validação de login
- proteção de páginas privadas
- menu dropdown do usuário
- navegação entre páginas
- layout responsivo
- design moderno dark/tech
